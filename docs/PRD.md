

《绩效核算系统 · 产品需求文档（P0）》
---------------------

### 1. 项目背景

医院（内科）绩效核算流程复杂、规则多、人工计算易错。  
目标是通过**导入标准模板 + 可视化配置规则 + 自动计算**，生成**科室绩效汇总发放表**，并支持按月留痕、对账、回看。

* * *

### 2. 使用对象

* 绩效核算人员（核心用户）

* 科室管理者（查看结果/对账）

* * *

### 3. 核心目标

* 导入院发绩效表 + 各类统计表

* 自动计算每位员工绩效

* 所有规则参数化（比例、权重、是否参与）

* 支持异常提示、对账、按月保存

* * *

### 4. 输入数据（P0）

| 表名     | 说明                    |
| ------ | --------------------- |
| 院发绩效表  | 项目名 + 金额（来源）          |
| 绩效发放名单 | 姓名 / 岗位 / 绩效分数        |
| 夜班统计   | 姓名 / 夜班数（医师护理合表，靠岗位拆） |
| 判读费    | 类别 / 姓名 / 金额          |
| 医师工作量  | 姓名 / 工作量 / 床日数 / 住院证数 |
| 护士工作量  | 姓名 / 分数 / 抽血量         |
| 手动录入   | 工作量分配、进修产假补贴等         |

* * *

### 5. 输出结果

* 《科室绩效汇总发放表》（按人）

* 明细钻取（每人每一项）

* 对账表（来源 vs 分配）

* 异常清单

* * *

### 6. 核心规则摘要（业务定稿）

* **夜班**：按夜班数分，合表靠岗位拆

* **判读费**：化验70%医师，30%护理池

* **床补**：2/3医师（床日数），1/3护理池

* **护理池**：
  
  * 权重=护理工作量分数
  
  * 护士长=平均分×1.4
  
  * 分数=0排除

* **医师汇总池**：
  
  * 权重=工作量
  
  * 工作量=0 → 0.8（门诊医师）

* **科室盈余**：
  
  * 主任15%
  
  * 护士长5%
  
  * 剩余按绩效分数
  
  * 排除进修/产假/请假

* **分摊综合手术室成本**：仅对账，不发个人（内科）

* * *
