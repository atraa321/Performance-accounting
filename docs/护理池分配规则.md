# 护理池分配规则

本规则用于固定“护理池”的获取与分配逻辑（当前实现口径）。

## 护理池来源项目

以下项目金额进入护理池（NURSING_POOL）：

- 床补的护理部分：`床补 * 1/3`（参数 `bed_subsidy_nurse_ratio = 0.3333333333`）。
- 护补：`NUR_SUBSIDY` 全额。
- 中医特色护理：`TCM_NURSING` 全额。
- 体检中心分配：`CHECKUP_CENTER_ALLOC` 全额。
- 判读费中的化验/检验部分：`READING_FEE * 30%`（参数 `lab_nurse_ratio = 0.3`，仅当判读类别包含“化验/检验”时）。
- 科室盈余的护理部分：
  - 先扣除科主任比例与护士长比例（直接发放）。
  - 剩余按医师/护理绩效权重拆分，护理部分进入护理池。
- 进修产假补贴（手动设置部分）：
  - 通过“手动池调整/手动池补录”表录入 `pool_code = NURSING_POOL` 的金额。

## 护理池分配规则

护理池按“护士权重”分配给护理人员：

- 权重基础：护士工作量表 `score`（仅使用 `score > 0` 的记录）。
- 护士长权重：`平均护士分数 * 1.4`（参数 `head_nurse_score_coeff = 1.4`）。
  - 若护士长不在护士工作量表中，仍按上述规则补入权重。
- 按权重比例分配护理池总额，分配金额总和应等于护理池总额。

## 备注

- 判读费类别为合并单元格时，导入会向下继承上一条类别。
- 护理池分配在“所有护理池来源金额入池之后”执行，避免少分配。
