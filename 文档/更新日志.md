# 更新日志

## 2026-01-21

### 🗑️ 批次删除功能

**新增功能**: 批次管理页面支持删除批次

#### 后端

- **新增接口**: `DELETE /runs/{run_id}`
- **功能**: 删除批次及所有相关数据
- **安全限制**: 已锁定批次无法删除

#### 前端

- **新增按钮**: 操作列添加删除按钮
- **二次确认**: 删除前显示确认对话框
- **友好提示**: 成功/错误提示

#### 删除范围

删除批次时会同时删除：
- ✅ 所有原始数据（7个表）
- ✅ 计算结果（汇总、明细）
- ✅ 对账数据
- ✅ 质量检查数据
- ✅ 批次本身

#### 使用方法

```
批次管理 → 操作列 → 删除 → 确认
```

**注意**: 
- ⚠️ 删除后无法恢复
- ⚠️ 已锁定批次不显示删除按钮

### 🎯 数据导入规则优化

**修改文件**: `app/calc/importer.py`

#### 新增功能

1. **排除金额为 0 的项目** ✅
   - 导入"院发绩效表"时，自动跳过金额为 0 或 None 的项目
   - 减少无效数据，提高计算效率

2. **排除合计行** ✅
   - 自动识别并跳过项目名为"合计"、"总计"、"小计"、"汇总"的行
   - 避免合计数据被当作独立项目处理

#### 影响

- ✅ 提高数据质量
- ✅ 减少无效计算
- ✅ 优化结果展示
- ⚠️ 导入记录数可能减少（正常现象，过滤的是无效数据）

#### 示例

**修改前**:
```
导入 100 条记录（包含 15 条金额为 0 的项目 + 1 条合计行）
```

**修改后**:
```
导入 84 条有效记录（自动过滤 16 条无效数据）
```
