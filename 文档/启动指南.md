# 项目启动指南

## 🚀 快速启动

### 一键启动（推荐）

```powershell
.\启动.ps1
```

这个脚本会自动：
1. ✅ 检查 Python 和依赖
2. ✅ 初始化数据库（首次运行）
3. ✅ 安装前端依赖（首次运行）
4. ✅ 启动后端服务（端口 8001）
5. ✅ 启动前端服务（端口 3000）
6. ✅ 自动打开浏览器

### 停止服务

```powershell
.\停止.ps1
```

## 🔧 手动启动

如果需要手动控制，可以分别启动：

### 启动后端

打开 PowerShell 终端 1：

```powershell
cd "d:/我开发的项目/科室绩效核算v2"
python3 -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8001
```

### 启动前端

打开 PowerShell 终端 2：

```powershell
cd "d:/我开发的项目/科室绩效核算v2/frontend"
npm run dev
```

## 🐛 常见问题

### 1. Python 路径不正确

如果您的 Python 不在 `D:\python38\python.exe`，请编辑 `启动.ps1`：

```powershell
$PYTHON = "你的Python路径"
```

### 2. 端口被占用

```powershell
# 查找占用进程
netstat -ano | findstr :8001
netstat -ano | findstr :3000

# 停止进程（替换 <PID> 为实际进程 ID）
taskkill /PID <PID> /F
```

### 3. 依赖安装失败

```powershell
# 后端依赖
pip install -r requirements.txt

# 前端依赖
cd frontend
npm install
```

### 4. 数据库错误

```powershell
# 重置数据库
Remove-Item perf_calc.db
D:\python38\python.exe -m alembic upgrade head
```
