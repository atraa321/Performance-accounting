## ⚠️ 注意事项

### 执行权限

如果遇到执行权限问题：

```powershell
# 设置执行策略（管理员权限）
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

# 或者临时允许
powershell -ExecutionPolicy Bypass -File .\测试后端.ps1
```

### 路径问题

- 脚本假设从项目根目录或脚本目录执行
- 如果路径不对，请调整脚本中的路径变量

### 端口冲突

- 测试脚本会检查端口占用
- 如果端口被占用，请先停止相关服务
