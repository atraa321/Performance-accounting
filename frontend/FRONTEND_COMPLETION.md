## 🚀 启动指南

### 1. 安装依赖

```bash
cd frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问：http://localhost:3000

### 3. 确保后端服务运行

```bash
# 在项目根目录
uvicorn app.main:app --reload
```

后端地址：http://localhost:8000

## 📖 使用流程

### 完整工作流程

1. **创建批次**
   - 访问"批次管理"页面
   - 点击"新建批次"
   - 填写月份、科室名称
   - 提交创建

2. **导入数据**
   - 点击批次进入详情页
   - 点击"导入 Excel"
   - 选择 Excel 文件
   - 等待导入完成

3. **验证数据**
   - 点击"验证数据"
   - 查看验证结果
   - 处理错误和警告

4. **配置规则**（可选）
   - 访问"规则参数"页面
   - 调整计算参数
   - 保存配置

5. **执行计算**
   - 返回批次详情页
   - 点击"执行计算"
   - 等待计算完成

6. **查看结果**
   - 查看汇总表
   - 查看异常列表
   - 查看对账表

7. **导出数据**
   - 点击"导出 Excel"
   - 下载结果文件
